⚙️ How It Works

  1. 9 AM UTC Daily: GitHub Actions wakes up and runs your autonomous agents
  2. Researcher Agent: Fetches new blogs from Notion (Status = "Ready for Social")
  3. Strategist Agent: Decides which platforms to post on and what angle to take
  4. Editor Agent: Creates platform-specific posts
  5. CEO Agent: Reviews quality and approves/rejects
  6. Poster Agent: Publishes approved content and updates Notion
  7. Logs: All agent activity is logged back to Notion for review

🚀 Manual Trigger

Want to test the agents right now without waiting for the daily schedule?

  1. Go to your GitHub repo → Actions tab
  2. Click "Daily Autonomous Agents" workflow
  3. Click "Run workflow" → "Run workflow"
  4. Wait 2-3 minutes for agents to complete
  5. Check Notion for new log entries
